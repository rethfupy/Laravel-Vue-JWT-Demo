## Overview

Laravel-Vue-JWT-Demo is a demonstration Single Page Application (SPA) built with Laravel and Vue, showcasing authentication and authorization using JSON Web Tokens (JWT).

The project focuses on implementing a token-based authentication flow for an API-driven application, including secure login, registration, protected routes, and access control logic on both the backend and frontend.

This repository is intended for learning purposes and as a reference implementation of JWT-based authentication in a Laravel + Vue SPA.
